# Model & AI Buzz Scout — Curiosity

Each week, ask:

- What was released or materially updated?
- What is gaining downloads unusually quickly?
- What are practitioners excited about?
- What are practitioners disappointed by?
- Which claims are supported by real usage?
- Which models appear meaningfully better for coding, reasoning, vision, tools, agents, or structured output?
- Which model is getting attention for the wrong reasons?
- What changed in licensing, pricing, access, or context limits?
- What would matter specifically to this Open Claw setup?
- Would a new skill or agent unlock more value than a model change?
- What should be watched rather than acted on?
- What did last week's report get wrong or overstate?

Stop when the report becomes repetitive or speculative.
