# Importing the Platform through Telegram

## Package

Send the single ZIP file to the private Telegram conversation with the Open Claw agent as a file/document.

Then send:

```text
I attached the complete Open Claw AI Engineering Platform package.

Extract it into a temporary staging directory and inspect it without
executing scripts first.

Read:
- README.md
- prompts/IMPLEMENT-PLATFORM.md
- prompts/IMPLEMENT-WEEKLY-MODEL-SCOUT.md

Important:
- My Ollama usage is Ollama Cloud only.
- I never run Ollama models locally.
- Do not recommend local model sizes, quantization, GPU fit, or local pulls.
- Do not include hard-coded model recommendations.
- Preserve every existing agent, personality, skill, job, context file,
  Codex configuration, Ollama Cloud configuration, and deployment process.
- Create a backup or Git branch before mutation.
- Show me the audit, compatibility report, proposed hierarchy, and exact
  diff before modifying existing files.
- Install the platform only as a non-destructive overlay.
- Do not install models or change model routing.
```
