# Product Discovery

Use before material product changes.

Produce:
- target user
- jobs-to-be-done
- workflow map
- capability-integrity matrix
- information architecture
- domain rules
- edge states
- acceptance criteria
- out-of-scope list

## Completion output

Return:
- findings
- changes or recommendations
- evidence
- unresolved risks
- explicit pass/fail status where applicable
