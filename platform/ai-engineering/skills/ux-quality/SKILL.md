# Ux Quality

Review the complete user journey.

Check:
- action hierarchy
- navigation
- copy
- form behavior
- error recovery
- destructive actions
- mobile usability
- cognitive load

## Completion output

Return:
- findings
- changes or recommendations
- evidence
- unresolved risks
- explicit pass/fail status where applicable
