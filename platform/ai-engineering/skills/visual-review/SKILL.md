# Visual Review

Render screenshots and critique them.

Required viewports:
- 390x844
- 768x1024
- 1440x1000

Review:
- hierarchy
- spacing
- alignment
- typography
- color
- density
- overflow
- long content
- realistic data

## Completion output

Return:
- findings
- changes or recommendations
- evidence
- unresolved risks
- explicit pass/fail status where applicable
