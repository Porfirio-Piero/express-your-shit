# Accessibility Standard

Required:

- keyboard-operable core workflows
- visible focus
- semantic landmarks
- labels for inputs
- accessible names for icon controls
- dialog focus trap and return focus
- reduced-motion support
- color is not the only state indicator
- sufficient contrast
- logical heading order
- status announcements where appropriate
- 44px mobile touch targets

Use automated scans, but do not treat automated scans as sufficient.
